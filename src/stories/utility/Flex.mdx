# Flex

- `.u-flx-shn`：`flex-shrink: 0`を適用
- `.u-flx-ctr`：`justify-content: center`を適用（主軸方向の中央寄せ）
- `.u-flx-y-ctr`：`align-items: center`を適用（交差軸方向の中央寄せ）
- `.u-flx-y-b`：`align-items: baseline`を適用（交差軸でベースライン揃え）

詳細は`/src/styles/utils/_flex.scss`をご確認ください。
