# Frames

詳細は`/src/styles/utils/_frames.scss`をご確認ください。

# AspectRatio

- `.u-asp-*`：`aspect-ratio`を適用
- 対応サイズ:`sq` `s` `m` `ys` `ym` `yl` `ppr` `yppr` `brz` `ybrz` `xbnr` `ybnr` `fbnr` `hbnr` `rbnr` `wsbnr`

# AspectRatioRotate

- `.u-asp-rt-*`：横長から縦長へ`aspect-ratio`を変換
- 対応サイズ:`s` `m` `l` `ppr` `brz`

# AspectRatioRiverse

- `.u-asp-rv-*`：縦長から横長へ`aspect-ratio`を変換
- 対応サイズ:`s` `m` `l` `ppr` `brz`

# borderRadius

- `.u-brd-r-*`：全ての角に`border-radius`を適用
- `.u-brd-r-t*`：上辺の角に`border-radius`を適用
- `.u-brd-r-b*`：下辺の角に`border-radius`を適用
- `.u-frm-r-*`：frameの内側にも`border-radius`を適用
- 対応サイズ:`s` `m` `l` `r`

# filter

- `.u-flt-gry`：子要素にグレースケール効果を適用
