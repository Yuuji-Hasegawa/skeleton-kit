# Display

- `.u-dsp-*`：`display`プロパティを指定 (`blc: block` `ibc: inline-block` `flx: flex` `ifx inline-flex` `tbl: table`)
- `.u-pc-{none|only}`:メディアクエリでPC表示の制御（非表示 or 表示限定）
- `.u-pos-r`:`position: relative`を適用

詳細は`/src/styles/utils/_display.scss`をご確認ください。
